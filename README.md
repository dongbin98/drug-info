# Drug_Info
약물의 부작용이나 오남용 방지를 위해 약학정보원에서 제공하는 약물 정보 조회를 할 수 있는 프로그램입니다.   
Selenium을 이용하여 약학정보원에 로그인 후, 필요한 정보를 Jsoup을 통해 파싱하여 텍스트 파일로 저장합니다.   

## Environments
- Windows 10
- Java

## Skill
- Jsoup
- Swing
- Selenium
